## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/LeoScripts/php-wj

# Access
$ cd php-wj

# Install dependencies
$ composer install

# Rum container Docker Mysql
make up

# Create Tables in database
# use script db.sql in your database

# Run the project
```
